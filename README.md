# PWmap
Этот проект представляет собой интерактивную карту мира Perfect World, реализованную на Leaflet.js.
Возможности:

Переключение между обычной и 3D-картой
Фильтрация и отображение ресурсов и монстров по категориям и тиру
Поиск и центрирование по координатам
Современный тёмный интерфейс
Для запуска откройте index.html в браузере.
Все данные хранятся в папке data, тайлы — в tiles и tiles_3d
